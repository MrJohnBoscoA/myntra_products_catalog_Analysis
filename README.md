# Myntra Product Recommendation System

## Overview
Analyzes Myntra product catalog to uncover pricing trends, color preferences, and gender demographics. Implements recommendation engine using cosine similarity.

## Features
- **Data Analysis**: Price distribution, color trends, gender preferences
- **One-Hot Encoding**: Feature engineering for categorical variables
- **Recommendation Engine**: Cosine similarity-based product recommendations
- **Insights**: Most expensive color groups, market segmentation

## Technologies
- Python 3.x
- Pandas, NumPy for data manipulation
- Scikit-learn for similarity metrics
- Jupyter Notebook

## Dataset
- Product catalog with price, color, gender, and category data
- CSV format with product features

## How to Run
```bash
jupyter notebook myntra_products_catalog_Analysis.ipynb
```

## Key Insights
- Price variation by color group
- Gender-specific product trends
- Feature-driven recommendation scores
